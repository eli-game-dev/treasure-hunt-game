# treasure-hunt-game

משחק מבוסס חידות חברתי המשלב תחרות וחווית משחק בתלת מימיד.
 
 השחקנים יפתרו מספר חידות אשר כל חידה תוביל לחידה הבאה\,
 והחידה האחרונה מובילה למטמון.
 
 הראשון שפותר את כל החידות ומגיע למטמון הוא המנצח.
